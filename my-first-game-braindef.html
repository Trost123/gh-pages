My first game - BrainDef

Back in 2012, after reading a lot of success stories on flashgamedev.ru I decided to try creating a game myself.

This was my first experience of programming. I used Adobe Flash (which uses ActionScript3 as a scripting language). I used this amazing tutorial to learn how to create a basic shooter game in Flash, and then just remixed it a little.

In this game you can find references to my favourite games of the time: Team Fortress 2, C&C, Wormix.

You can play BrainDef on Kongregate.
