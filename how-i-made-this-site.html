How I made this site - no programming, no hosting bills

Today I'm going to tell you how I quickly made this website using Publii and hosted it for free on GitHub Pages.

My goal was to quickly develop and host a simple, yet good-looking portfolio\blog site for free. After some research I found Publii - a no-database CMS, that allows to easily upload to GitHub Pages for a free hosting. And it features a nice set of free themes!

Publii is a free desktop app that allows you to edit and manage your site's content using intuitive user interface. No html\css knowledge required.

(Although, I do know how build sites from scratch, I think using a ready solution is smarter than hand-crafting your site from scratch.)

That's it. I could go more in-depth about setting it up, but there are already plenty of guides on it.
